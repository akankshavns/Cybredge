CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation and formatting of a document written in HTML or XML

Type of Css

  1. Inline css
  2. Internal Css
  3. External Css
  • Inline CSS:
    • Styles are added directly to an HTML element using the style attribute. This is useful for unique, one-off style adjustments but makes the code harder to maintain if used extensively.
  • Internal CSS:
    • Uses a <style> element within the <head> section of the HTML document. This applies styles to that specific page only.
  • External CSS:
    • The recommended method for most projects, which involves linking a separate .css file to the HTML document using a element. This allows a single CSS file to control the layout and appearance of multiple web pages at once.
